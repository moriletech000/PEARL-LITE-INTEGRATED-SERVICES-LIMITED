# Pearl-Lite Integrated Services Limited — Website

A luxury, high-converting static website for **Pearl-Lite Integrated Services Limited**, a full-spectrum agro-industrial company based in Epe, Lagos State, Nigeria.

## Project Structure

```
├── index.html      # Main HTML file
├── style.css       # All styles and responsive layout
├── main.js         # Animations, interactions, form handling
├── logo.png        # Company logo
└── README.md
```

## Sections

- **Hero** — Full-screen with animated floating particles and a cinematic farm background
- **Marquee Strip** — Scrolling ticker of all core services
- **About** — Company story, layered image cards, animated stat counters
- **Services** — 6-card grid covering all business offerings
- **Products** — 4 export product showcases with image zoom on hover
- **Why Us** — Dark feature section with glassmorphism cards
- **CTA Banner** — Conversion-focused call to action
- **Contact** — Address, phone, business hours + working enquiry form
- **Footer** — Full brand info and quick links

## Features

- Fully responsive — mobile, tablet, desktop
- Fixed navbar that turns solid on scroll
- Mobile hamburger menu with close button
- Scroll-triggered reveal animations (Intersection Observer)
- Animated particle system in the hero
- Counter animation on stats
- Contact form with success feedback
- No dependencies — pure HTML, CSS, and vanilla JS
